INSTRUCTIONS before running TIMEOUT TIMER
*****************************************
1. In order for the ringtone to sound when the timer finishes, user must finish setting up the emulator in Settings.
2. Simply go into Settings in the emulator, there will be a suggestion titled: "Finish setting up your device".
        Simply click "FINISH SETUP" button and skip all the steps if desired.
